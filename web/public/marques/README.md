# Marque emblems

Drop official manufacturer emblem SVGs here, named by marque slug:

    porsche.svg   ferrari.svg   audi.svg   lamborghini.svg   ...

They render automatically as the badge next to each vehicle in the list
(see src/app/ui/Navigator/MarqueBadge.tsx). Until a file is present, a
monogram stands in. Use square-ish, transparent-background SVGs; they're
drawn at ~20px. Use officially licensed / press-kit assets only.
